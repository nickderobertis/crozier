## Style convergence

`Parameter` has no `style` field (`src/openapi.rs:361-398`), so serde intentionally
discards both `deepObject` and `spaceDelimited`; neither selects a newly added
production branch. `unsupported_query_styles_are_discarded_before_standard_serialization`
(`tests/generation.rs:3870-3898`) renders unstyled, deep-object, and space-delimited
specs and asserts both styled raw clients equal the unstyled client, then pins the
shared dict annotation and query entry. This tests the convergent contract rather
than claiming nonexistent style-specific serialization.
